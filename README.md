## The Platform (Today)

NovxX is currently a full-stack EV intelligence web application. Built with React, Node.js, Express, and MongoDB.

### Live Features

- **Battery & Efficiency Analyzer** — calculate real-world EV range based on battery level, temperature, terrain, driving style, vehicle type, AC usage, and load
- **Smart Insight Engine** — logic-based insights comparing your current trip against your historical data (Level 1 + Level 2)
- **Visual Analytics Dashboard** — range history charts, efficiency trends, driving condition breakdowns using Recharts
- **User Profile System** — vehicle type selection (Sedan, SUV, Compact EV, Hatchback, Truck) that adjusts calculator logic
- **NovxX Concept Vehicle Page** — the philosophy, cockpit vision, battery intelligence, and future mobility roadmap

### Coming Soon

- Charging Station Locator with real-time availability
- GPS-integrated maps
- AI-powered route planning
- Proprietary ML models for battery prediction

## The Story

I started this journey with a simple goal: to build something meaningful for the future.

I am a self-taught developer from Tank, KPK, Pakistan. No formal computer science degree. No big team. Just a clear vision, a laptop, and the determination to build something real.

I became deeply interested in Electric Vehicle technology — not just the software side, but the actual engineering. I have been studying EV fundamentals from the ground up:

- ✓ **How Cars Work** — completed

Every feature I build on this platform is connected to what I am learning about real EV engineering — battery management systems, energy consumption, thermal behavior, efficiency optimization. The software reflects the physics.

This is not a tutorial project. This is not practice. This is the beginning of a company.
