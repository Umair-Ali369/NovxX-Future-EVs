### Design Changes

**Colors:**
- `bg-gray-900` / `bg-gray-800` → `#091413` / `#0F1F1D` throughout
- Chart grid lines: `#374151` → `#ffffff08` (subtler on dark background)
- Chart accent: `#3b82f6` → `#44ACFF` (consistent with brand)

**Stat Cards:**
- Added icons: Average Range 📏, Most Used Condition 🛣️, Total Calculations ⚡
- Added hover border effect: `hover:border-[#44ACFF]/30`
- Value color: `text-blue-600` → `text-[#44ACFF]`

**Recent Calculations Table:**
- Header row: plain `<th>` text → `text-xs uppercase tracking-wider text-gray-500`
- Efficiency column: plain text → colored badge (green/amber/red matching value)
- Row borders: `border-gray-700` → `border-white/5`
- Added row hover: `hover:bg-white/[0.02]`
- Added `overflow-x-auto` wrapper for mobile
